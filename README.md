## Description

This is a Node.js web application for creating and maintaining notes. It uses technologies such as MongoDB for data storage, Express.js for server-side functionality, EJS for templating, and Passport.js for Auth0 user authentication. The project is designedÂ to be deployed on Google Cloud for scalability.

## Features

- Create, edit, and delete notes.
- Secure user authentication with Auth0.
- Utilizes a MongoDB database for efficient data storage.
- Modern CSS and HTML design.
- Scalable deployment on Google Cloud.
